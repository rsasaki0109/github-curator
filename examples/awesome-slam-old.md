# Awesome SLAM

## LiDAR SLAM
- [cartographer](https://github.com/cartographer-project/cartographer) - Google's SLAM system
- [LOAM](https://github.com/laboshinl/loam_velodyne) - LiDAR Odometry and Mapping
- [LIO-SAM](https://github.com/TixiaoShan/LIO-SAM) - Tightly-coupled LiDAR Inertial Odometry
- [FAST-LIO](https://github.com/hku-mars/FAST_LIO) - Fast LiDAR-Inertial Odometry

## Visual SLAM
- [ORB-SLAM3](https://github.com/UZ-SLAMLab/ORB_SLAM3) - Visual-Inertial SLAM
- [VINS-Fusion](https://github.com/HKUST-Aerial-Robotics/VINS-Fusion) - Multi-sensor state estimator

## Learning-based
- [DROID-SLAM](https://github.com/princeton-vl/DROID-SLAM) - Deep Visual SLAM
- [PythonRobotics](https://github.com/AtsushiSakai/PythonRobotics) - Python sample codes
- [DeepSLAM](https://github.com/voicelabs/DeepSLAM) - Deep learning SLAM framework
